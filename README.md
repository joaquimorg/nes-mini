### Connection to Raspberry Pi
| Controller  | Raspberry Pi  |
| ------------ | ------------ |
|Black   - GND   |  Pin 6 - GND |
|White  - GND   |  N/C |
|Yellow - SCK   |  Pin 5 - SCL |
|Green  - SDA   |  Pin 3 - SDA |
|Red     - VDD   |  Pin 1 - 3.3V |
